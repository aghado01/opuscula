# codex-scientiae

A sample of structured, long-form reference material — a *bound-knowledge corpus* optimized for agent
retrieval. Every document here was produced by the **codex-membrane** restoration pipeline (see the
companion [codex-membrane](../codex-membrane/) opusculum): machine-extracted from PDF, repaired
chunk-by-chunk to faithful, LaTeX-consistent markdown, then published into the collections below.

Navigation is **telescoping** — start at [CONTENTS.md](CONTENTS.md), which points to each collection's
root, and follow the nested `CONTENTS` down to chapters and sections. The index is the single source of
truth, not this README.

## Collections

- **[codices/](codices/CODICES.md)** — long-form textbooks (Bishop 2006, Grimmett 2006, McLachlan 2000).
- **[compendia/](compendia/COMPENDIA.md)** — theme-ordered paper collections (bars, mapper, ph).

> A curated snapshot for portfolio review — a subset of a larger private corpus, under development and
> not production-certified. Output format follows the membrane's `STANDARDS`: dollar-delimited LaTeX,
> pure semantic syntax, markdown tables, reference sidecars.
